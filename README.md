En este repositorio utilizamos Python 3 con Django Framework para nuestra primer app en la clase de Lenguajes de programacion
detalles del trabajo acontinuacion 

Implementar una aplicación Web que administre una libreta de contactos.
El contacto tiene como atributos el identificador del nombre del contacto, el teléfono del contacto, los correos electrónicos, las referencias a redes sociales y vínculos entre los contactos. Por ejemplo: el contacto 1 es hermano del contacto 3.
1. Se debe crear una base de datos (Cualquiera) para almacenar estos datos.
2. Se debe crear una página HTML para mostrar la libreta de contactos y realizar su respectiva administración
3. Se debe crear los comandos para agregar, editar y borrar los contactos de la libreta.
4. Se debe poder agregar más correos electrónicos, referencias a redes sociales de los contactos y/o vínculos entre los contactos
